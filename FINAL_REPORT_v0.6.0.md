## 技术架构

- **前端**: Vanilla JavaScript (ES6+) + HTML/CSS
- **后端**: FastAPI + Uvicorn
- **数据库**: SQLite + SQLAlchemy ORM
- **调度器**: APScheduler 3.10.4
- **部署**: Systemd服务 + Nginx反向代理
- **数据验证**: Pydantic模型
